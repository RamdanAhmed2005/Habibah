# سيرة حبيبة محمد الذاتية - موقع ويب 🌸

## إزاي تضيفي صورك الحقيقية؟
1. افتحي مجلد `img`.
2. حطي 3 صور باسم بالظبط:
   - `photo1.jpg`
   - `photo2.jpg`
   - `photo3.jpg`
3. احفظي وافتحي ملف `index.html` تاني، الصور هتظهر مكان المربعات الوردي تلقائيًا.

(لو مفيش صور لسه، هيفضل يظهر تصميم وردي حلو مكانها لحد ما تضيفيها).

## إزاي تشغلي الموقع؟
كفاية تفتحي ملف `index.html` بأي متصفح (كروم/إيدج..) دبل كليك وهيشتغل على طول.

## محتويات المشروع
- `index.html` → الصفحة الرئيسية
- `css/style.css` → التنسيقات والألوان والحركات
- `js/script.js` → التفاعلات (السكرول، النموذج، الأزرار)
- `img/` → مكان الصور
